Firebase er ikke konfigureret endnu. Følg disse trin for at komme i gang — det tager ca. 5 minutter.

1. Opret Firebase projekt Gå til console.firebase.google.com → "Add project" → følg guiden
2. Opret Firestore database I dit projekt: "Build" → "Firestore Database" → "Create database" → vælg "Start in test mode"
3. Hent dine nøgler Project Settings (⚙️) → "Your apps" → "</> Web" → registrer app → kopiér firebaseConfig
4. Indsæt nøglerne i denne fil Åbn autocare.html i Notepad og erstat værdierne i FIREBASE_CONFIG øverst i filen
5. Host filen gratis Gå til tiiny.host → upload filen → få en URL du kan åbne fra alle enheder

Se README.md for detaljeret vejledning med screenshots
